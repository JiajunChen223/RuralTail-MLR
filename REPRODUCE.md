# Reproducing RuralTail-MLR

This document records the paper-protocol reproduction path for TALC.

## Protocol

The machine-readable protocol is:

```text
configs/protocol/paper_talc.yaml
```

The single public reproduction chain is:

```text
prepare data -> train score sources -> evaluate paper protocol -> make paper tables
```

The public score sources are:

```text
bce
focal
asl
talc
```

TALC uses ASL score learning plus frequency-group SoftF1 and the one-sided tail
recall term. Fixed and group operating variants are evaluated after training.
Global and classwise threshold searches are diagnostic outputs only and are not
part of the public method set.

Score sources are trained for up to 30 epochs with validation-mAP early
stopping. The best checkpoint is selected by validation mAP.

## Setup

```bash
conda env create -f environment.yml
conda activate ruraltail-mlr
pip install -r requirements-dev.txt
python tools/check_environment.py --require_timm
```

For the MLM-R18 carrier:

```bash
pip install -r requirements-mamba.txt
python tools/check_environment.py --require_mamba_ssm --require_causal_conv1d
```

## Metadata

Each dataset uses the same metadata interface:

```text
images_index.csv
labels_clean.csv
fixed_split.csv
class_mapping.json
class_frequency.csv
head_medium_tail.json
```

The H/M/T grouping is computed from training-label frequencies only.
China-MAS-50k uses split seed `20260425`; Agriculture-Vision-2021 uses field-id
group splitting with seed `20260501`.

## Train Score Sources

```bash
GPU_LIST=0,1,2 bash scripts/train_score_sources_china.sh
GPU_LIST=0,1,2 bash scripts/train_score_sources_agv.sh
```

Both scripts support `JOB_INDICES` for partial reruns and write checkpoints to
`outputs/runs`. Each training run also writes a single-run fixed `0.5` (`-F`)
test result; paper-facing carrier and method tables are generated by the
unified evaluation command below.

## Evaluate Operating Boundaries

```bash
GPU_LIST=0,1,2 bash scripts/evaluate_paper_protocol.sh
```

This command builds `artifacts/eval_manifest_seed2026.json`, predicts validation
and test sets, writes formal fixed `0.5` (`-F`) results, selects H/M/T
thresholds on validation only for formal group (`-G`) results, and freezes
those thresholds for test. Global and classwise thresholds are written as
diagnostic CSV files only.

## Paper Tables

```bash
bash scripts/reproduce_paper_tables.sh
```

Outputs are written under:

```text
artifacts/paper_tables/
```

The paper-facing columns are `mAP`, `Macro-F1`, `Tail mAP`, `Tail P`, `Tail R`,
and `Tail F1`.
